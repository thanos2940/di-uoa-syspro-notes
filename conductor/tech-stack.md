# Technology Stack: System Programming Web Notes & Quiz

## 1. Frontend
- **Language:** HTML, CSS, JavaScript (Vanilla)
- **Rationale:** A simple, dependency-free approach that ensures fast load times, easy maintainability, and longevity without the overhead of heavy frameworks.

## 2. Backend & Deployment
- **Platform:** Vercel
- **Architecture:** Serverless static site delivery via GitHub integration.
- **Rationale:** Vercel provides seamless continuous deployment from GitHub and exceptional global performance for static assets.

## 3. Tooling & Scripting
- **Language:** Python
- **Use Case:** Utility scripts for data processing, formatting, and file manipulation (e.g., `replace_dup.py`, `scratch_fix.py`).
- **Integration:** These scripts can either be run locally for pre-processing or integrated into the Vercel build process.
